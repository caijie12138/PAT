1056. 组合数的和(15)

Standard
作者

给定N个非0的个位数字，用其中任意2个数字都可以组合成1个2位的数字。要求所有可能组合出来的2位数字的和。例如给定2、5、8，则可以组合出：25、28、52、58、82、85，它们的和为330。

输入格式：

输入在一行中先给出N（1<N<10），随后是N个不同的非0个位数字。数字间以空格分隔。

输出格式：

输出所有可能组合出来的2位数字的和。

输入样例：
3 2 8 5
输出样例：
330

//Created By Caijie,Senior Student of BJTU
#include<iostream>
using namespace std;
int main(){
    int num;
    cin >> num;
    int nums[num];
    int sum = 0;
    for(int i = 0;i<num;i++)
        cin >> nums[i];
    for(int j = 0;j<num;j++)
        for(int k = 0;k<num;k++) {
            if (k == j)
                continue;
            sum += nums[j]*10 + nums[k];
        }
    cout << sum;
}
